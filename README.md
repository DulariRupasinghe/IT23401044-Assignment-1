**Converter from Singlish to Sinhala**

This application translates Sinhala written in English letters (Singlish) into genuine Sinhala script. 
Just input a sentence in Singlish, and the software will convert it automatically in real time while maintaining English words, punctuation, and numbers.
It supports a variety of phrase structures, including as instructions, questions, statements, and colloquial idioms.

**How to Use**
1. Launch the application.

   Open the application in your browser by going to the given URL (such as `https://www.swifttranslator.com/`).
   
2. Type in Singlish Text

   Enter your Sinhala sentence in the input field using English characters (Singlish).
   
3. View the output in Sinhala

   The output section will automatically display the translated Sinhala content.
   
4. Translation in Real Time

   The program refreshes the translation in real-time while you input continually.
   
5. Particular Situations

   Punctuation, dates, numerals, abbreviations, and English terms are all retained.

   Repeated words, slang, and colloquial language are also addressed.
   
6. Copy or Utilize the Outcome
    You can copy the Sinhala text for your own use after the translation is finished.
